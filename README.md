# AI Lead Qualification System

This system is a Python-based engine designed to evaluate business leads and improve sales efficiency. It analyzes specific data points—such as the time since a last enquiry and budget status—to automatically categorize leads into "Hot," "Warm," or "Cold" statuses.

---

##  Project Roadmap
This project tracks the evolution of the engine through daily milestones. To maintain technical clarity, **every milestone folder includes its own dedicated README.md file** detailing the specific logic, improvements, and technical notes for that stage.

*   **[Day 1: The Foundation](./day_01/README.md)**  
    Initial build focusing on core categorization logic.
*   **[Day 2: Production Hardening](./day_02/README.md)**  
    Refined version featuring modular architecture, error
    handling, and professional logging.

## 🚀 How to Navigate
1.  **Milestone Folders**: Browse the `day_xx` folders to see how the system has evolved over time.
2.  **Dedicated Documentation**: Read the README.md within each folder for deep-dive technical explanations of that day's code.
3.  **Source Code**: Each folder contains the standalone Python script for that specific version of the qualifier.
